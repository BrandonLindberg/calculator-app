# C++ Calculator App

Simple command-line calculator that has a basic menu, and supports basic arithmatic with handling for invalid inputs.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Use C++ IDE of choice to open and run project from repository

Instructions for using the software:

1. Run project in IDE
2. Use command-line for calculation

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* I used CLion as an IDE, with CMake to build the project files.
